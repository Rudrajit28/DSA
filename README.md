🧩 DSA

A personal, auto-updating log of Data Structures & Algorithms problems solved in C++

📖 About

This repo is my running scratchpad for DSA practice. Every time I solve a problem — mostly from LeetCode — it gets pushed here automatically, one folder per problem.

No polish, no cherry-picking. Just an honest, growing record of the reps.

🎯 Why


Build consistency through daily/weekly practice
Create a personal reference for recurring patterns (two pointers, sliding window, binary search, DP, etc.)
Track progress over time in one place


🛠️ Tech


Language: C++
Primary source: LeetCode


🤝 Using this repo

Feel free to browse, fork, or reference these solutions for your own practice. Pull requests aren't expected since this is a personal log, but suggestions/issues are welcome.
